# Сборщик контента (HTTP клиент + сервер)

Реализация задания: HTTP клиент и сервер для сбора информации из интернета с сохранением в LocalStorage.

## Функциональность

- **Сервер (Node.js + Express)**:
  - Хранит соответствие ключевых слов и URL.
  - Возвращает список URL по ключевому слову.
  - Эмулирует скачивание контента с передачей прогресса (chunked response).

- **Клиент (Vanilla JS + Fetch)**:
  - Ввод ключевого слова, получение списка URL.
  - Выбор URL и скачивание контента через сервер.
  - Отображение прогресса загрузки (проценты, статус).
  - Сохранение контента в LocalStorage.
  - Просмотр сохранённого контента офлайн.
  - Обработка всех ошибок с выводом пользователю.
